Write debug output to console.

Write debug message and wait for "time" milliseconds. If "endl" is true, the console will print the text and end this line and begin a new line with the next function call. If endl is false, the console will stay in the current line and the next text will be attached to the current string in this line. See class description for more information!

Parameters

timeWait-time after printing. Standard wait-time is 1sec.

Write formatted data output to console.

Write formatted data and wait for "time" milliseconds. If "endl" is true, the console will print the formatted data and end this line and begin a new line with the next function call. If endl is false, the console will stay in the current line and the next text will be attached to the current number in this line.
