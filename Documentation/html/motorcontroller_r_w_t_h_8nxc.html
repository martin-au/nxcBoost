Detailed Description

Background task and high level routines to control motors.

Define NO_CONTROLLER_TASK_SERVICE if you do not need non blocking motor commands

Function Documentation

void motorEmergencyStop ( bool  shutDown = true)

Stop (all) motors and shut down motor controller.

This function will immediately brake the motors and shut down the motor-controller-task. If you set shutDown to false the program will continue but some motor control tasks will not shut down immediately.
