void rgb2hsl ( int &  rgb[],
int &  hsl[],
int  admax = 1024 
)

Convert RGB to HSL colour space.

Convert red, green, blue raw values to hue, saturation, lightness values to draw easier distinctions between different colors.

hue: 0 - 360
0 -> red
60 -> yellow (for nxt 10-15, problems with orange, yellow)
120 -> green
180 -> light blue
240 -> blue
300 -> pink?

saturation: 0% - 100% (nxt color sensor sat values are usually low)
0% -> chromatic (grey)

lightness: 0% - 100%
0% -> dark (black)
100% -> bright (white) (throw away all measurements that have light values over 75%)

More information: http://en.wikipedia.org/wiki/HSL_and_HSV

Array indices for hsl:
#define INPUT_HUE 0
#define INPUT_SAT 1
#define INPUT_LIGHT 2

Parameters
rgb[]An array containing at least three raw color values. See Color sensor array indices (nxc api).
hsl[]An initialized array containing at least three elements.
admaxThe maximum of rgb values (255 or 1024)
string strtok ( const string  str,
const string  del 
)

Split string into tokens separated by an delimiter.

A sequence of calls to this function split str into tokens, which are sequences of contiguous characters separated by a string of a delimiter.

On a first call, the function expects the token-string as argument for str, whose first character is used as the starting location to scan for tokens. In subsequent calls, the function expects a "NULL" string and uses the position right after the end of last token as the new starting location for scanning.

The function returns in every single call one token. If there are no more tokens left, the function returns a "NULL" string. If str or del contains no character, the function returns also "NULL".

Parameters
strThe string containing the tokens or "NULL" continue with current string
delDelimiter between tokens-
Returns
string Current token.
