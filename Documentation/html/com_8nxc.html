void initCom ( const byte  outbox = 0,
const byte  inbox = 0 
)

Initialize the communication module.

Use this to initialize the communication module. The paramters are set to default values. Standard send-time = 0 ms, polling-time = 25ms, timeout = 1s. You can initialize the com-module with standard in/outbox.

Parameters
outboxSet default outbox for sending messages. (0-8)
inboxSet default inbox for sending messages. (0-8)
void purgeMailbox ( byte  mailbox)

Clean inbox.

This functions removes all messages from the the specified mailbox inbox. If the mailbox number is negative, then all mailboxes will be cleaned.

Parameters
mailboxInbox 0-9 or any negative number.

Try to receive an number.

Wait until theres a number in the specified inbox. You may set a timeout if you want to wait a longer or shorter period. If there is no message in the given timeout, the function will return false Note that this function is also designed for simplicity: receiveString(myMsg);

Parameters
inNumberThe received number. Number should be a floating-point number.
inboxSee nxc api or inbox constants.
timeoutTimeout in ms if theres no message in the inbox.
pollingTimeThe time to wait between every request in ms.
Returns
False if there is an error. (timeout)

Wait until theres a string in the specified inbox. You may set a timeout if you want to wait a longer or shorter period. If there is no message in the given timeout, the function will return an error string. Note that this function is also designed for simplicity: myMsg = receiveString();

void setComTime ( const unsigned long  sendTime)

Set default send-time.

You may set a new default send time. Standard wait-time is 0. (NXT-PC-HMI checks every 20 ms Msg-Box) The send-time is used after every send function.

Parameters
sendTimeTime in milliseconds.
